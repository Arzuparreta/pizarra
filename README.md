# pizarra
El objetivo de este proyecto es solventar la necesidad que naturalmente todos nosotros tenemos de hostear tu servicio ultra-simple de pizarras, editables y visualizables en tiempo real desde tus dispositivos. (made with Node.js)

La idea de las pizarras es que previsualizen la mayor cantidad de tipos de archivos posible con todas sus extensiones (imágenes, texto y código, audio en baja resolución, etc...) y links o referencias a cualquier otro tipo de archivo que no pueda ser previsualizado.
